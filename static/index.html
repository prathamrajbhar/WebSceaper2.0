POST /api/search

Perform a search on Google or Bing.

curl -X POST http://localhost:8002/api/search \
     -H "Content-Type: application/json" \
     -d '{"query": "openai", "engine": "google"}'
POST /api/scrape

Extract clean text content from any public URL. 

curl -X POST https://webscrapeer-0dd75353a1e5.herokuapp.com/api/scrape \
     -H "Content-Type: application/json" \
     -d '{"url": "https://example.com"}'
